# Customer Personality Analysis
## Маркетинговий аналіз клієнтської бази роздрібного магазину

### Про проект
Повний цикл аналізу бази з 2232 клієнтів роздрібного магазину. Мета — визначити хто генерує найбільше доходу, як працюють маркетингові кампанії і які сегменти клієнтів найбільш цінні для бізнесу.

**Інструменти:** Python (pandas, numpy) · Power BI · DAX

---

## Загальні висновки та інсайти

**Інсайт #1 — Принцип Парето підтверджено**
27% клієнтів (High Value) генерують 65% всіх витрат. 46% клієнтів (Low Value) приносять лише 7% доходу. Сегментація базується на перцентилях Total_Spending: High Value > $1,044 (вище 75-го перцентилю), Mid Value $69–$1,044 (між 25-м і 75-м перцентилем), Low Value < $69 (нижче 25-го перцентилю).

**Інсайт #2 — Пріоритетність каналів**
Хоча Catalog має найвищий відгук на кампанії (17,09%), він генерує лише 21% покупок. Store домінує за покупками (46%) але показує найнижчий відгук на кампанії (6,05%). Catalog варто використовувати як інструмент залучення а не основний канал продажів.

**Інсайт #3 — Ефект останньої кампанії**
Last Campaign показала 14,87% відгуку — вдвічі більше за середнє (~7%). Варто проаналізувати що саме зроблено інакше (механіка, таргетинг, timing) і масштабувати цей досвід.

**Інсайт #4 — Маркетинг працює але не туди**
66% всіх відгуків на кампанії генерує High Value сегмент — маркетинг утримує лояльних клієнтів але майже не активує пасивних. Low Value сегмент (46% клієнтів) реагує лише на 11% кампаній.

**Інсайт #5 — Активні клієнти витрачають вдвічі більше**
Клієнти які реагували на кампанії витрачають $1,001 проти $458 у тих хто ігнорував — різниця 118%. Важливо: це кореляція а не причинність — датасет не містить даних про витрати до і після кампаній.

**Інсайт #6 — Вікова група 40-49 — недооцінений сегмент**
Середні витрати $502 — найнижчі серед усіх вікових груп. Групи 30-39 ($683) і 60+ ($693) витрачають на 35% більше. Потребує окремого дослідження — питання асортименту, цін або комунікації.

**Інсайт #7 — Wines та Meat — основа бізнесу**
Wines ($679K) — майже 50% всіх витрат. Meat ($373K) — стабільний другий пріоритет. Разом ці дві категорії складають понад 77% всіх витрат.

## Бізнес-рекомендації

**1. Утримання High Value клієнтів — пріоритет №1**
Програма лояльності направлена на топ 27% клієнтів окупиться швидше ніж будь-яка інша ініціатива. Втрата навіть невеликої частини цього сегменту критично впливає на загальний дохід.

**2. Окрема стратегія для Low Value сегменту**
Стандартні кампанії не працюють на цю аудиторію. Потрібен інший підхід — нижчий поріг входу, інші категорії товарів або інший канал комунікації.

**3. Масштабувати досвід Last Campaign**
14,87% відгуку — результат який варто вивчити детально. Відповідь на питання що зроблено інакше може подвоїти ефективність наступних кампаній.

**4. Catalog як інструмент залучення**
Не скорочувати Catalog як канал — але переосмислити його роль. Висока чутливість до кампаній (17,09%) робить його ідеальним інструментом для таргетованих маркетингових активацій.

**5. Дослідити вікову групу 40-49**
Найнижчі витрати серед усіх вікових груп — це або недооцінений сегмент або аудиторія з іншими пріоритетами споживання. Потрібне окреме дослідження причин.

**6. Wines та Meat — пріоритет для акцій**
Фокус промо активностей на топ категорії дасть максимальну віддачу від маркетингового бюджету.

## Дашборд 1 — Customer Insights

### Мета
Відповісти на два бізнес-питання: хто наш клієнт і хто генерує найбільше доходу.

### Структура
- **3 KPI картки:** Total Revenue, Avg Customer Spending, Total Purchases
- **Фільтри:** Customer Segment, Income Group, Age Group
- **4 графіки:** сегменти, вік, категорії товарів, канали продажів

### Ключові інсайти

**Принцип Парето підтверджено:**
27% клієнтів (High Value сегмент) генерують 65% загальних витрат. Low Value сегмент складає 46% клієнтів але приносить лише 7% витрат. Mid Value — 27% клієнтів і 29% витрат, найбільш збалансована група.

**Найцінніший вік:**
Клієнти вікових груп 30-39 та 60+ витрачають найбільше ($683 та $693 в середньому відповідно). Найменш активна група — 40-49 років ($502) — потенційно недооцінений сегмент.

**Товарні категорії:**
Wines — беззаперечний лідер ($679K, близько 50% всіх витрат). Meat на другому місці ($373K). Gold ($98K) — попри нішевість, випереджає всі харчові категорії окрім м'яса: Fish ($84K), Sweets ($60K), Fruits ($59K).

**Канали продажів:**
Офлайн магазин домінує — 46% всіх покупок. Онлайн канал займає 33%, каталог — 21%. Каталог залишається живим каналом — кожна п'ята покупка.

## Дашборд 2 — Marketing Campaign Performance

### Мета
Оцінити ефективність маркетингових кампаній та проаналізувати як різні сегменти клієнтів реагують на маркетингові пропозиції.

### Структура
- **3 KPI картки:** Campaign Engagement Rate, Total Campaigns Accepted, Marketing Impact on Spending
- **Фільтри:** Customer Segment, Income Group, Age Group
- **4 графіки:** ефективність кампаній, витрати responded vs not responded, відгук по сегментах, відгук по каналах

### Ключові інсайти

**Ефективність кампаній:**
Last Campaign — 14,87% відгуку, вдвічі вище середнього (~7%). Campaign 2 — аномально низький відгук (1,34%).

**Вплив маркетингу на витрати:**
Responded — $1,001 середні витрати. Not Responded — $458 середні витрати. Різниця 118% — але це кореляція а не причинність.

**Реакція по сегментах:**
High Value генерує 66,17% всіх відгуків. Low Value — лише 11,28% попри 46% частку клієнтів.

**Реакція по каналах:**
Catalog — 17,09% відгуку (найвищий). Web — 8,31%. Store — 6,05% (найнижчий попри домінування в покупках).
